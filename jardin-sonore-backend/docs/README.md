1. intention et flux macro ;

## Quand mettre a jour cette doc

Mettre a jour ces fichiers quand :

- une commande CLI change de comportement observable ;
- un seuil metier, une option ou une convention d'usage change ;
- un module applicatif gagne un nouveau flux utilisateur ;
- une note `.codex` devient une verite produit/technique durable et doit etre promue en doc versionnee.
